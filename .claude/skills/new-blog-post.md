# Skill: create a new blog post

## When to use
When asked to create a new article, insight, or technical post.

## Steps
1. Ask for: title, language (en/es), topic summary, tags
2. Create file at: content/posts/YYYY-MM-DD-[slug].mdx
3. Add frontmatter with all required fields
4. Write a strong opening paragraph that hooks the reader
5. Structure: intro → problem → approach → insight → takeaway
6. For technical posts: include a code block or diagram where relevant
7. Keep tone: professional, direct, technical but approachable

## Output
A complete MDX file ready to publish with no placeholders.
